구매 플로우를 앱에 붙이는 거고, 핵심은 아래야.

메인 화면 활동 소식 위에 새 섹션 추가

대표 이미지 1장 + 토지 구매 버튼
버튼 누르면 상세 구매 페이지로 이동
상세 구매 페이지 입력 항목

이름, 국가, 주소, 연락처
종교 1개 선택
스타일 옵션을 드롭다운으로 선택
세계관/종교, 입지/지형, 분위기/구성, 시각효과, 특별 서비스
결제 버튼 클릭 시 구매 완료 문구 노출
“***님의 토지가 구매되었습니다. 3~4주 후 증명서 발송”
“이용료는 불우한 이웃 지원에 사용” 안내
구매 데이터 처리

사용자 계정 정보에 저장
관리자 이메일로 주문 데이터 전송
관리자는 AI 이미지 생성 + 증서 출력/실링왁스 + 우편 발송
증서 템플릿

천국 영토 소유 증서 (Certificate of Heavenly Estate) 형식
소유권/축복 메시지/발행일/서명 문구 포함
종교별 명칭 반영 (예: 불교=극락, 기독교=천국, 이슬람=잔나 등)
다국어 발송

최소 2장: 종교 근원 언어 + 해당 국가 언어
예시처럼 종교별 언어 매핑(아랍어/히브리어 등) 적용
